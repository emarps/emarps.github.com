Why we use the Normalize CSS Script

There are many reasons why we use Normalize CSS Script wheen working with CSS. The most basic answer is that normalizing css evens out the differences between various browsers. This allows for consistency and preserves useful browser defaults instead of erasing them. Most designers don't start from scratch so Normalize or a CSS reset is almost required in modern frontend development.

What is Semantic Markup

Semantic markup instroduces meaning to a web page rather than the style or presentation. The biggest reason why one should use this markup is so you can provide additional information about the document. This helps with making clear and consistent communication throughout your document.
